    《人人都爱经济学》知识要点

    发表于 2017-01-22 更新于 2024-06-21 分类于 经济 阅读次数： 阅读次数：

    本文为阅读《人人都爱经济学》之后总结的知识要点

    总共分为20个章节

    第一章 经济学到底是什么

    • 经济学：关于资源如何配置的学问

      • 能让人更好地理解社会、人生、世界
      • 人的欲望无穷，而资源稀缺，因此“资源配置”很重要
      • 资源配置：生产什么、如何生产、为谁生产
      • 资源配置的目标：经济学的效率 （帕雷托效率）—— 最好的状态：如果要使一某个社会成员变得更好，则必须使得另一个社会成员变得更差

    • 资源配置的方法：

      • 计划经济

        • 必然失败

      • 市场经济

        • 手段是市场
        • 价格机制

    • 经济学理论

      • 微观经济学

        • 一个消费者、厂商、市场是如何行为的

      • 宏观经济学

        • 一个国家甚至全世界的经济史如何运行的

        • 如果出了问题怎么办

    第二章 经济学如何看世界

    三种思维方式

    • 边际考虑：

      • 概念：只考虑最后一个增加的一个所引起的变化，从而判断事情的整体本质

      • 例子：

        • 农民种粮食 - 边际产量递减 - 工资是边际产量
        • 吃东西 - 边际效用递减
        • 物以稀为贵

    • 机会成本考虑：

      • 概念：由于做这件事情而放弃的其他用途中价值最大的那一个的价值
      • 体会：在一定时间内，只有做利益最大化的事情，才是获利，否则都意味着亏损
      • 对于资源的配置非常重要

    • 均衡考虑：

      • 概念：边际好处（收益） = 边际坏处（成本）的状态
      • 均衡是一种稳态，一种确定，方便人们观察研究
      • 均衡意味着变化缓慢
      • 均衡是边际考虑的结果
      • 均衡意味着各种力量达到妥协 - 如果不能做到均衡，事情就不会结束 - 为了达到有意义的结局，必须调整自己的行为，同时不能顾此失彼

    第三章 经济学的基石：经济人假设

    • 经济人假设／理性人假设：

      • 假设每个人／自然人、厂商，都是在给定约束下追求自己利益的极大化

    • 利他 = 利己：

      • 利己不损人的前提下
      • 利他：客观上：利他。主观上：利己，使自己快乐。

    • 利己 = 利他：

      • 每个人都只知道如何极大化自己的利益，且都不知道对方的利益为何物

      • 当每个人仅仅从自己的利益极大化出发时，有一只看不见的手，在引导着他，使他在追求自己利益的过程中，对他人的利益也最大。

      • 每个人从自己的利益出发去行事，要比他从他人的利益出发去做事，结果对他人更好

      • 举例：市场- 竞争的压力使得他必须对别人好，这就是市场的力量

        • 看不见的手 = 市场

    • 义的本质 = 利

      • 人都是逐利的
      • 君子小人都喻于利

    第四章 经济学的新方法：博弈论

    • 背景

      • 博弈论告诉你怎么跟人打交道，如何参透别人的心思
      • 新古典经济学的标志：
      • 马歇尔经济学 ——假定市场是完全竞争的，任何一个人的行为对其他人都没有影响，其他人的行为也不会影响自己的行为
      • 但无法解释寡头政治——于是采用博弈论

    • 博弈论

      • 分析人民之间的行为相互影响的条件下，每个人如何决策

    • 经济学最重要的三个概念：

      • 需求
      • 供给
      • 纳什均衡

    • 博弈的分类：

      • 完全信息静态博弈

        • 纳什均衡
        • 囚徒困境与看不见的手的结果是相反的，这是因为博弈的双方不知道对方的选择
        • 完全：局中人对于对方的战略和各自的支付是清楚的
        • 静态：只能做一次选择，或双方不知道对方的选择
        • 举例：囚徒困境

      • 完全信息动态博弈

        • 举例：郭嘉遗计定辽东

      • 不完全信息静态博弈

        • 不完全：博弈的各方都有一些信息是自己知道而对方不知道的 -——依据概率行事——先验概率
        • 举例：是否与一个自称武林高手的人过招

      • 不完全信息动态博弈

        • 动态：可以做多次决策，或知道对方的行动之后调整
        • 举例：空城计

    • 博弈论可以把事情的本质揭露得更清晰，有助于匡正长期以来的似是而非

    第五章 供给和需求

    • 价格由什么决定：（马歇尔提出的）

      • 供给（也就是生产的重要要素）： 劳动，资本，土地，企业家才能
      • 需求：边际效用，主观的感受

    • 供给和需求都是函数

      • 供给：价格和供给量的函数，反应生产者的行为 —— 价格越高，供给量越多，因为会获得更多的利益
      • 需求：价格和需求量之间的关系，反应消费者的行为 ——价格越高，需求量越小
      • 均衡价格：必然会存在一个价格，使得供给量等于需求量，此时对双方都是最好的价格

    • 供求决定价格：

      • 决定需求：本身的价格、替代品的价格、消费者的偏好。。。
      • 需求定律：当其他条件不变时，一件商品本身的价格越高，对他的需求量久越少，反之则越多
      • 决定供给：生产的技术、是否能赚钱、投入品的价格。。。
      • 供给定律：商品本身的价格越高，厂商的供给量就会越多，反之就越少

    • 价格是市场的核心

      • 看不见的手：买卖双方互相看不见，但能够相互作用形成价格
      • 价格把所有有用的信息都包含在里面，把所有无关的信息都过滤掉

    • 价格机制：

      • 价格上下运动，以让市场的供给量和需求量相等的过程

    第六章 消费者理论

    • 消费者理论：

      • 在收入有限的情况下，如何实现自己的效用极大化

        • 收入有限 ——资源稀缺

      • 让花在每一种商品上的最后一单位货币，比如最后一块钱，所获得得效用都相等

      • 边际考虑

    • 效用的单位：

      • 效用的单位无关紧要
      • 因为用边际分析的方法，不同单位不过是大小不同而已
      • 把效用的单位增大或减小，对结果没有任何影响

    • 周围环境对消费行为的影响

      • 价格：（比如可乐和电影票，本来是均衡的）

        • 现象：可乐价格下降，购买增多
        • 原因：本来可乐和电影票是均衡的（最佳状态），也就是花在两种物品上的最后一单位货币的效用上相等的。现在可乐的边际效用 = 最后一听可乐带来的效用 ／ 价格。价格下降，则边际效用增加。为使它们重新平衡，就必须买更多可乐，来让可乐的边际效用重新与电影票的边际效用相等。

      • 收入：

        • 收入影响的是货币的边际效用，使它减少。而货币的边际效用 = 每种商品最后一单位所获得得效用 ／ 价格。

      • 所以只有让商品的边际效用也减少才能重新均衡。因此需要增加购买。

    第七章 厂商理论

    • 厂商：

      • 企业、生产者
      • 目的：获得经济利润
      • 看作函数：自变量：投入品（原材料、劳动，资本） 因变量：产量
      • 生产函数：只取产量的最大值

    • 产量：

      • 特点：边际产量递减
      • 含义：在保持其他投入的量不变的条件下，让某一种投入要素的量不断增加，这种要素增加引起的产量增加越来越少

    • 最佳配比：

      • 需要多种要素才能生产时，必须确定没中要素的投入量，或者最佳配比
      • 原则：购买每种要素，使得最后一块钱所产生的边际产量相等

    • 机会成本计算：（3部分的总和）

      • 会计成本
      • 企业自有资金成本，也就是本来投入在企业的钱，如果去收利息，可以收多少
      • 企业所有者的自有时间的机会成本。比如如果老板去机关上班会挣多少钱
      • 这三部分的总和，可以看作机会成本

    • 总收益：（4部分组成）

      • 工资：劳动者
      • 利息：资本家
      • 地租：土地所有者 （土地的特征是需求量大，而供给固定）
      • 正常利润：总收益（会计利润） - 上述三者。因为企业家投入了企业家才能
      • 经济利润：会计利润 - 机会成本。这部分叫超额利润，归资本家或者股东所有。

    • 总成本

      • 固定成本：土地之类的

      • 可变成本：劳动力

      • 亏损：价格 < 平均固定成本 + 平均可变成本

        • 但只要产品的价格仍然大于平均可变成本，则可继续销售，以弥补平均固定成本的亏损

    • 最佳临界点：

      • 边际好处 = 边际坏处
      • 边际好处：新增加一单位产量所增加的毛收入，叫边际收益／边际好处
      • 边际坏处：为增加这一单位产量必须付出的总成本，叫做边际成本／边际坏处（注意！只与可变成本有关！因为固定成本与产量无关，也与当前的决策无关）
      • 边际成本递增

    第八章 竞争和垄断

    • 完全竞争：

      • 是一种理想的、假想的市场结构

      • 条件：

        • 有无数多个买者和卖者
        • 信息是完全的，不用广告，消费者也知道产品的品质、性能等
        • 产品是没有差别的
        • 进入和退出这个市场是自由的

      • 结论：任何一个消费者生产者都是市场价格的接受者，都没有能力创造价格或者改变价格

        • 如果要改变，也必须所有人同时作出改变

        • 此时的价格对消费者来说是最好的

          • 消费者剩余：消费者愿意支付的价格和实际支付的价格的差额，是一种消费者福利（只存在于主观）

        • 但是对于生产者，不会吃亏，也没有便宜可占。因为此时 经济利润 = 会计利润 - 机会成本 = 0

          • 因为如果有更好的机会，则大家都会知道，都会蜂拥至另外一个产品的销售上，于是会计利润再次等于机会成本

      • 完全竞争的市场结构中，人们的竞争大道了一种非人格化的状态，竞争是悄无声息的，人们察觉不到

        • 此时厂商只需要决定生产什么，而不需要考虑定价

    • 垄断：

      • 概念：

        • 某种产品的市场上之有唯一的一个供给者
        • 绝对的垄断是不存在的
        • 在长期，由于有定价权，则一定有经济利润

      • 分类

        • 自然垄断：当一种产品的平均生产成本随着产量的增加而减少时，由一家厂商提供，其平均成本最低，由多家厂商提供反而就高。所以由一家垄断经营时，价格有可能是最低的。

          • 比如：公共事业比如天然气，微软的os等

        • 行政垄断：政府通过行政力量，组织其他厂商进入，只让自己指定的厂商运营的垄断

          • 比如：国家的铁路，邮政，电信和银行等

      • 自然垄断和行政垄断没有必要同时存在

        • 如果有自然垄断，则没必要有行政垄断
        • 有了行政垄断，则区分不清是行政垄断还是自然垄断了

      • 自然垄断的价格控制：

        • 价格听证会
        • 可以高于边际成本，使得厂商愿意经营，政府也不用补贴；否则按边际成本，厂商无法盈利，则政府需要补贴
        • 反垄断法

      • 垄断的好处：

        • 没有竞争压力，可以进行创新研发，改进品质和服务
        • 完全竞争的企业，则没有动力继续创新和研发了

      • 垄断最终是不好的：

        • 垄断者的低价，并不是他们的仁慈，要回报社会，而实际上是想剥夺消费者剩余，自己赚更多的钱而已。

    • 价格战：

      • 价格歧视：飞机上的乘客，除头等舱和普通座位之外，还有价格上的不同

        • 目的：使得乘客更多，只要保证价格仍然高于平均成本即可

      • 打折：比如电影院的学生价

        • 目的：增加学生源，只要价格仍然高于影院的平均成本，影院的利润就会增加

    第九章 垄断竞争和寡头

    • 垄断竞争

      • 概念

        • 最常见的真实市场

        • 既有完全竞争的特征，又有垄断的特征

          • 完全竞争

            • 这个市场厂商非常多，有可替代性，比如川菜馆
            • 信息相当充分，大家都互相知根知底

          • 垄断

            • 每个厂商的产品都于其他厂商的产品有细微的差别，这些差别对某些消费者而言构成垄断，比如喜欢更喜欢吃不辣的川菜，就会去某家川菜馆——忠诚度

      • 经济利润：

        • 短期内可能会有经济利润

          • 价格定低，会增加一些刻骨，但不会太多，因为有垄断（这就是它与完全竞争的差别）
          • 价格定高，可以在出某些新品的时候

        • 长期不会有经济利润

          • 因为信息是充分的，只要还有利润，就会有新的厂商进入，直到不存在经济利润

          • 这就是垄断竞争和垄断的主要区别

            • 垄断：短期内可能亏损，但由于有定价权，所以不会亏损，长期会有经济利润
            • 垄断竞争：没有经济利润

      • 广告：

        • 由于垄断竞争者之间是人格化的，相互能感受到对方是对手，因此要影响消费者的偏好
        • 广告是最核心的营销手段之一
        • 广告增加了对产品的需求，但是也会增加生产的总成本
        • 判断广告的最佳数量：边际利润 = 边际成本

      • 优缺点：

        • 缺点：缺乏效率，因为存在资源的浪费，实际生产量太少。比如饭店有很多空座位，加油站有很多闲置的加油机
        • 优点：可以满足不同层次消费者的偏好，消费者拥有充分的选择权

    • 寡头

      • 概念：

        • 如果一个市场是只有很少的几家厂商
        • 厂商之间互相牵制，彼此关联，不但要根据自己的情况，还要考虑他人的反应

      • 主导厂商理论：

        • 市场最大的一家，产量最大，最有名气
        • 主导厂商是价格领导者，市场价格由它决定
        • 比如微软

      • 没有主导厂商的市场：

        • 厂商之间的影响是相互的
        • 价格是由厂商之间的互动决定的
        • 这个时候必须用博弈论

    第十章 国内生产总值：GDP

    • 概念：

      • GDP是指一个国家在一年内，所生产的群不最终产品（包括劳务）的市场价格的总和

    • 概念的5个解读

      • GDP是一个价格概念，是市场价格的和

        • 因为如果拿其他单位去做度量，都没有可比性，而价格具有共同的单位
        • GDP本来是一个福利的概念，凡是增加了福利的东西，都包括在GDP里——这是GDP的缺陷

      • GDP是最终产品的价格的和

        • 最终产品就是用于最终消费，比如收获了面粉，卖给中间商做成面包胚，再卖给面包店加工成面包，最后出售，那么只计算最终面包的钱
        • 计算方式：计算每个环节的附加值。

      • GDP是一个生产概念，不是消费概念

        • 只要是新生产的东西，都可以算进去，不管卖出了多少钱。比如一个东西生产的价格是100，但卖了80，此时GDP按照100计算
        • 股票等金融市场交易的金额，不能计算进去，因为没有生产东西，只是交易

      • GDP是一个流量概念，不是存量

        • 流量是一个时间段内的量，存量是一个时间点上的量，比如财富是存量，而GDP是流量。
        • 只看本年度生产出来的东西，不是看卖出的时间——为了避免重复计算

      • GDP是一个地理概念

        • 中国的GDP暂不包括港澳台
        • 只要在大陆境内生产的产品，都算作中国的GDP，即使是外资企业产生的

    • 人均GDP

      • 表示人民富裕程度
      • 人均GDP= 总GDP／总人口 ≠人均可支配收入
      • 因为还要减去交给政府的税收

    • 绿色GDP

      • 把污染的价格从GDP里减掉
      • 但计算困难

    • GDP的弱点

      • GDP很难反映实际福利
      • 只计算合法行为的价格，违法和犯罪行为、或者本身不违法，但是有逃偷税的行为的，不能计入GDP

    第十一章 总需求：经济繁荣的根本

    • 经济危机：

      • 东西生产太多，卖不出去，造成生产下降，工人失业，人们对未来失去渴望
      • 产生原因：相信马歇尔的古典经济理论，即市场完全有效——价格是灵活的，价格是调整供需的标杆
      • 解决方法：凯恩斯理论

    • 凯恩斯理论

      • 经济萧条的原因：人们不愿意买东西，即总需求不足

      • 原因：

        • 物品的边际效用递减
        • 资本的边际效率递减
        • 流动偏好：人们总会始终保留一部分钱在手里，不会全部花出去

    • 萨伊定律：

      • 一个东西在生产的同时，对它的需求也生产出来了
      • 但是经济危机显然推翻了这个结论，因为东西卖不出去了

    • 凯恩斯的有效需求理论

      • 由于以上三个原因，单单依靠家庭和企业的力量，需求一定是不足的，需要借助国家的力量
      • 政府发行国债，向民众借钱。民众不是没钱，只是不愿意花钱。政府发行国债并提供利息，民众就愿意借钱。政府拿钱去买东西，东西卖完了又可以开始生产，又开始雇佣员工，就业增加，市场恢复运转。

    • 启示

      • 政府是看得见的手，可以在一定程度上解决市场这只看不见的手所造成的经济困境
      • 市场不是万能的
      • 需求比供给更重要，消费比生产更重要

    • 凯恩斯核心命题：

      • 在短期，均衡的产量，即对于一个社会最佳的产量，是由总需求的水平决定的

    • 宏观经济政策：

      • 凯恩斯创立
      • 核心：在于估计均衡的产量或者GDP是多少，依据：总需求
      • 经济师繁荣还是萧条，最根本的原因是： 总需求。因此如果能拉动总需求，经济就会繁荣
      • 我们总是被教导应该成为怎样的人，可是从来没有人告诉我们，我们在本质上是什么样的人。——《蜜蜂的寓言》曼德维尔

    • 其他启示：

      • 了解正常人有虚荣之心，顺其自然，按照这个人性去诱导人的行为，我们的生活会更快乐，经济也会更繁荣

      • 不要一味指责虚荣。虚荣不过是喜欢美好的东西，更尊重自己，不想亏待自己罢了。

      • 奢侈，在某种意义上，也是人类进步的主要标志和动力之一。奢侈也是文明的一种表现。

    第十二章 财政、税收与财政政策

    • 财政

      • 前言

        • 财政政策：政府如何理财——本质：提供公共物品
        • 经济政策：财政政策+货币政策
        • 所有政策都是短期的，都是权宜之计，不是长期的

      • 概念

        • 财政，全称”公共财政“，指政府的财政收支活动

      • 职能

        • 提供公共物品，比如：基础教育、环保、基本医疗——就像“海边礁石上的灯塔”

          • 特点：
            • 非竞争性：不管你怎么消费，都不影响别人消费，比如：国防
            • 非排他性：没法阻止别人消费，比如：阻止别人呼吸

        • 解决经济萧条

          • 在此之前，政府的职能：提供国防；提供司法；提供公共设施——严守收支均衡
          • 经济危机之后，政府不严守收支均衡，而是发行国债，主动造成财政赤字，以增加总需求

      • 两个方面

        • 收入

          • 来源：税收：政府强制向居民和企业征收的货币或者实物

          • 税收：

            • 特征：

              • 强制性：采用强制手段——国家本质是暴力，政府是暴力的垄断者，即人民自愿放弃使用暴力的权利，而交由政府专门行使。
              • 无偿性：征税后，不用给任何回报。——不是一对一的有偿，是对整体有偿。即对于一个具体的人而言，你交不交税，交多少，跟你从政府得到多少利益没有直接对应关系。
              • 法律规范性：依据法律，预先规定好的，不是随意征收。

            • 作用：

              • 为政府供应公共物品提供财力支持

              • 调节总需求

                • 增税：降低总需求
                • 减税：增加总需求

        • 支出

          • 购买支出

            • 概念：政府直接花钱，得到相应的商品和劳务，比如国防支出

            • 对总需求的影响：

              • 直接，因为政府的购买也是总需求的一部分

          • 转移支付

            • 概念：政府作为中介，把部分钱转移到部分人手里，比如低保

            • 对总需求的影响：

              • 间接，不过最终会变成消费者或者生产者的消费和投资，从而增加总需求

    • 财政政策

      • 乘数效应：
        • 倍增效果，你的支出就是他的收入，所以最初的一笔支出可以引起数倍于原来的总需求。——四两拨千斤
      • 分类：

        • 扩张性

          • 概念：扩大总需求

          • 方法：

            • 减税：不直接增加总需求，只有通过消费者或企业增加自己的消费和投资才会引起。——政策时滞。但要慎用，因为减税之后，要增加就难了
            • 增加政府开支、发行国债：直接增加总需求，因此效果更明显，也更佳
        • 紧缩性
          • 概念：缩小总需求
          • 方法：增税，减少政府开支
      • 挤出效应：

        • 概念：当政府不断增发国债，增加支出时，经济中的储蓄被政府吸收过来了。企业的投资最终来自于家庭的储蓄。当政府拿走绝大部分时，企业不得不提高利率去争夺剩下的储蓄，而这会降低企业的投资。

        • 判别：当政府的扩张性支出引起利率提高时。

        • 条件：

          • 当经济中的失业不严重，接近于充分就业时，同时资源的闲置也不严重，挤出效应就会出现。
          • 因为此时经济景气，企业想要投资，就会增加利率，就会发生挤出，但不严重，所以总需求还是增加的；但如果已经接近充分就业时，政府再增加财政支出，就会加剧资源紧张。

    • 凯恩斯的宏观经济学：

      • 只关注短期，不关注长期

        • 短期：

          • 情况：经济繁荣和萧条交替出现，经济的主要问题是失业和通货膨胀
          • 解决的问题：如何防止经济过热，如何走出经济萧条

        • 长期：

          • 情况：经济将是平稳的，失业和通货膨胀都不会存在，因为没有人会失业一辈子

      • 所有的政策都是为了熨平短期内的经济波动

    第十三章 货币和银行

    • 货币

      • 概念

        • 货币是用来交换的媒介
        • 纸币：信用货币——是靠政府的信用支撑其价值，本身只是纸而已

      • 货币政策中的货币

        • 货币体系分成不同层次

        ​ 每个层次影响不同

        • 使用货币政策时，要注意不同层次的货币对经济的影响不同

      • 货币的三个层次（在中国）

        • 最低层次——M0：包括纸币和硬币，也就是现金／通货

          • 特点：
            • 最容易买到商品
            • 流动性最强
            • 对市场的冲击力最大

        • 中间层次——M1: 不仅包括M0，还有活期存款

          • 特点：流动性比M0小

        • 最高层次——M2: 不仅包括M1，还有定期存款和企事业单位的存款

          • 特点：广义货币；现在中央银行的货币政策，所关注的就是M2

      • 货币的价格 ／ 利率

        • 本身没有价格，因为一块钱就是一块钱，没有意义

        • 货币的价格 = 利率 = 利息／本金，表示借钱的难易程度

        • 货币的价格由需求和供给共同决定

          • 需求：企业和个人

            • 货币的作用：

              • 交换的媒介
              • 支付的工具
              • 价值储藏的手段

            • 财富的持有形式：

              • 货币
              • 股票
              • 房子
              • 。。。

            • 货币的需求是企业和个人对财富持有形式选择的结果

          • 供给：央行发行，相对固定

        • 利率的重要性——影响投资

          • 利率越高，投资的成本就越高，总需求减少，过热的经济可以冷却
          • 利率越低，投资成本越低，总需求增加，缓解经济不理想

    • 银行

      • 赚钱方式：

        • 提供借贷双方的中介，从中收取劳务费

      • 起因：

        • 借款人担心资金安全，也找不到需要借钱的人
        • 贷款人担心交易费用太大

      • 解决方法：

        • 银行以它的组织信誉和管理能力，成了信贷双方的中介
        • 一个人的钱实际上借给了无数个项目，平摊了风险
        • 每个项目经过风险评估，保证储户的钱相对安全

      • 作用：

        • 使得资金高效率地配置到了最需要它的地方
        • 交易因此增加——交易和投资对一个社会的繁荣和发展的必要性和重要性
        • 创造社会财富，也节省信贷双方的费用

    第十四章 中央银行与货币政策

    • 中央银行

      • 各国有各国的中央银行

        • 中国：中国人民银行
        • 美国：美联储

      • 中央银行与商业银行

        • 中央银行：

          • 不追求盈利，不接受个人和企业贷款；
          • 维护金融系统的安全稳定，防止金融危机的发生；
          • 促进经济增长，增加就业——靠货币政策

        • 商业银行：

          • 营利性机构，通过存贷业务和其他业务來赚钱维持
          • 怕 挤兑：存款人大批量地从银行取钱，银行无力应付，宣布破产，容易引发经济危机

    • 货币政策

      • 概念：

        • 让货币多点或少点，控制货币供给量
        • 这里的货币指M2

      • 货币的多少

        • 凯恩斯校正了货币的观念，认为货币多少，对实体经济及人们生活，会造成实在的影响
        • 货币是个先行因素
        • 金融，是现代经济的核心——邓小平

      • 影响方式：

        • 货币多 – 利率低 — 投资欲望小 —总需求降低
        • 货币少 – 利率高 — 投资欲望大—总需求增加

      • 3种货币政策

        • 存款准备金

          • 商业银行存在中央银行的钱——因为不允许商业银行将钱全部贷出去，一定要留一些；存着可以获得利息
          • 提高准备金率，商业银行的贷款就少了，利率就高了，投资就下降，总需求也下降
          • 即，反比

        • 贴现率

          • 商业银行向中央银行借钱的利息率
          • 中央银行钱不够了可以自己发行
          • 贴现率越高，商业银行借到钱就越不容易，就越不容易贷款，利率就高了，投资下降，总需求也下降
          • 即，反比

        • 公开市场操作

          • 公开市场：买卖国债的市场，是二级市场
          • 中央银行不能直接认购政府债券，因为如果这样，央行原本的贷款规模适当，现在突然要增发货币给财政部，就会引发通货膨胀。而不通过中央银行，而是直接公开市场，则份额不会改变
          • 央行从商业银行那里买入政府债券，则相当于“吐出货币”，商业银行的货币就多了，就可以发放更多贷款，投资增多，总需求加大，经济繁荣了起来
          • 在美国，基本上都依靠这一货币政策；但中国，这一政策没有前两个好用，因为公开市场规模还不大

        • 货币政策的调节作用

          • 扩张性：降低准备金，降低贴现，买进国债
          • 紧缩性：提高准备金，提高贴现，卖出国债

        • 货币政策与财政政策搭配使用：

          • 严重通货膨胀：

            • 紧缩性货币政策：压缩货币总量，抑制投资过热
            • 宽松的财政政策：增加政府开支——防止总需求下降过快，失业增加的负面后果

          • 严重失业：

            • 扩张性财政政策：增加政府开支，增加总需求
            • 紧缩性货币政策：控制货币闸门，防止总需求无限扩张，造成通货膨胀

      • 其他：

        • 央行还可以直接规定存贷款利率
          • 因为中国的商业银行都是国有银行
          • 但其他国家不是
          • 美国的花旗银行等都是私人银行

      • 央行的权利

        • 中国：受国家控制
        • 美国：私有机构，只对国会负责

    • 政策要有效，就要出其不意，如果被人猜到了意图，就会预先采取行为來对付政策的变动，政策就会无效

    第十五章 失业

    • 定义

      • 经济学里，把适龄的，愿意接受现行工资条件，但没有工作，且又在寻找工作的劳动力，称为失业
      • 自愿失业者：有劳动能力，但不愿意去工作
      • 下岗：严格意义上的失业
      • 失业率：失业人数占全体劳动力人数的百分比

    • 失业的原因

      • 自然失业：

        • 摩擦性失业：因为地区改变而暂时失业
        • 结构性失业：因为产业结构升级、技术更新换代而被淘汰掉的失业
        • 当 自然失业率 = 失业率，称这个社会为充分就业——政府的核心目标之一

      • 经济衰退或经济周期造成的失业

        • 东西卖不出去，总需求减少

    • 失业的好处：给就业者带来心理压力，迫使他们提高生产效率

    • 隐性失业：虽然在工作，但没有带来任何贡献，却阻碍了有能力的人进入企业

    • 解决失业的方法：

      • 最低工资法？——错误

        • 原因：劳动力的需求量和工资称反比，和劳动力供给成正比。
        • 提高了最低工资，则劳动力需求量降低，反而增加了失业

      • 帮助提高失业者工作技能

      • 教育

      • 提供信息：提供覆盖全国的、权威的劳动力供求信息系统

      • 解决歧视：比如人种歧视等

      • 社会保障制度：社会福利好，有的人自愿失业，因为拿的钱差不多

    • 解决失业，最终还是要靠经济增长，即始终保持并创造出总需求

    第十六章 通货膨胀

    • 定义

      • 价格总水平的上升。价格总水平，是所有商品价格的加权平均，加权的依据是商品的重要性和产量。

      • 注意：不是个别商品的价格上涨，而是很多商品价格上涨，并持续一段时间

      • 表示通货膨胀一般用：

        • CPI，消费者价格指数，记录城市居民用于购买代表性消费品的价格变动，并跟踪购买行为
        • 之所以没有用生产资料，是因为生产资料价格的上涨，最终也会反映在消费品价格上

    • 坏处：

      • 通货膨胀有益于债务人，而对债权人有害，它会把财富转移到债务人手中
      • 受益者：政府。因为政府发行国债，而通货膨胀，政府是债务人，受益

    • 通货膨胀的原因：

      • 总需求拉动

        • 总需求增加，但总共就那么多东西，而投给它们的货币不断增加，则通货膨胀

      • 成本推动

        • 比如石油，或者工资上涨

      • 惯性通货膨胀

        • 人们的预期，比如大家都预期明年价格会上涨5%，所以工资也要上涨5%，结果明年的价格就不得不上涨5%，这就是“自我实现的预言”

    • 治理的方法——堵住货币发行的闸门

      • 压缩总需求：

        • 紧缩性货币政策+宽松的财政政策
        • 紧缩性财政政策+宽松的货币政策

      • 实行工资和物价管制，控制成本上升

        • 但这个方法是临时性的

      • 经济衰退來控制价格上涨

        • 因为通货膨胀和失业率增加呈现周期性交替出现

    • 失业和通货膨胀，两个最主要的宏观经济问题

      • 失业比通货膨胀更严重！！因为会降低社会总产出，还可能酝酿社会动荡

      • 通货膨胀不一定是坏事，是双刃剑

        • 可以缓解资源稀缺带来的供求失衡
        • 因为在经济增长过程中，资源的约束越明显，比如对土地、水、石油，如果不允许涨价，则会造成这些稀缺资源的过度使用，影响可持续发展
        • 在3%以内的通货膨胀是能接受的

    第十七章 经济增长

    • 定义

      • 经济增长，是潜在GDP的增加
      • 是一个国家没有通货膨胀时的最大产出，也就是充分就业时的常量
      • 衡量一个国家的生产能力
      • 用GDP的增长率 或 人均GDP增长 来表示

    • 经济增长是长期问题

      • 人口以几何级增长，产量却以算术级增长，因此边际产量递减，应该控制人口
      • 亚当斯密很乐观，认为总体上会经历经济增长

    • 经济增长的四个因素

      • 自然资源

        • 一个国家的地理位置和占有的自然资源，比如水，石油等，得天独厚的优势

      • 物质资本

        • 机器设备，即用于生产的机器设备
        • 要想经济增长，必须使得资本投入增加的速度快于劳动投入增加的速度——资本深化

      • 技术进步

        • 概念：在资本和劳动不变的条件下还能使产量增加的东西——缓解了边际产量递减的速度

        • 技术的不同表现形式

          • 提高生产效率：酿酒技术
          • 提高福利：外科手术，民用飞机

        • 技术影响生产力

          • 怎么生产 – 机器，设备，生产方式
          • 怎么安排生产 – 劳动力分工，提高生产效率

        • 一项技术能在多大程度上促进经济增长，关键看它扩散的速度

          • 比如电视，手机……

        • 技术的核心——基础研究

          • 创新≠发明，创新是模仿，是对已有技术的非原创的改进，比如方便面，卡拉OK，游戏机。。。
          • 创新所起的作用，远不如发明创造
          • 基础研究需要大量的投入，且需要等上多年才能有实际效果

      • 人力资本

        • 概念：人的技能，组织性，纪律性
        • 比如日本人的组织性，纪律性和严谨，使得日本的人力资源很好
        • 人力资本／资源是通过教育和培训获得的

      • 制度

        • 制度本身的优越性和缺陷会影响经济的发展，比如市场经济制度本身，优于计划经济

    第十八章 国际贸易和比较优势

    • 国际贸易

      • 原因：做贸易比不做贸易要好

      • 解释：

        • 绝对优势

          • 亚当斯密提出
          • 如果一个国家生产某种产品的直接成本，也就是会计成本，比别的国家低，该国就在生产该产品上有绝对优势
          • 但是只能解释部分国际分工的原因

        • 比较优势

    • 比较优势

      • 概念：生产某种产品的机会成本低于别人

      • 李嘉图“比较优势理论”：

        • 即使一个国家样样都比别人强，都有绝对优势，也没有必要什么都生产；即使一个国家样样不如别人，每种产品都没有绝对优势，但肯定在某一种上有比较优势

      • 按照比较优势进行分工，照样可以增加各国的福利水平

    第十九章 自由贸易还是贸易保护

    • 概念

      • 自由贸易：商品、资金、人员在全世界范围内应自由地流动
      • 贸易保护：对贸易进行限制

    • 自由贸易好：

      • 才能让各国发挥比较优势，实现资源在全球的最佳配置

    • 贸易保护的借口

      • 自由贸易会增加失业？

        • 实际上没有，是改直接生产为间接生产，实际上提高了就业率

      • 保护幼稚产业？

        • 因为新兴，所以弱小，需要保护——错误。不需要保护，因为如果失败，则保护也没用
        • 高关税还会引发走私
        • 配额（把进口的数量分配给不同国家，决定着从每个国家进口的份额）也会引发腐败

      • 为了发展某些高科技产业，扶持有些高科技首先进入市场，必须限制国外产品的竞争？

        • 首先进入市场不见得就好
        • 对高科技没有定义，不一定就好
        • 初创时期成本大，后进入可以少走弯路

      • 自由贸易破坏了环境？

        • 环境问题在经济发展中不可避免
        • 比如非洲，他们宁愿忍受污染，也不愿意挨饿，因此把污染产业转移到非洲，对非洲暂时是有利的
        • 环境倒U曲线，即，在经济发展初期，会随经济的增长而污染，达到一定程度后，会开始治理

    第二十章 汇率和国际货币体系

    • 汇率

      • 汇率是什么

        • 用别国的货币表示本国货币的价格

      • 如何决定汇率

        • 外汇市场

        • 供求决定

          • 想用人民币兑换美元，则中国公民是需求者，美国公民是供应者

      • 汇率受价值影响

        • 人民币对美元的汇率提高 == 人民币升值，则用相同的人民币可以买到更多的美元，美元就便宜了，但买的人就多了，需求就增加了；反过来美元持有者用相同的美元兑换到的人民币就减少了，就不愿意兑换了，则供应减少了。如果汇率过高，美国人就不愿意兑换了，中国人就会降价，直到美国人愿意买为止。
        • 美国人认为人民币的汇率被压低了，所以就很喜欢买中国的东西——导致贸易顺差，和大量外汇储备

      • 顺逆差

        • 贸易顺差：货币流入，但物品流出，别人喜欢买我们的东西（说明我们的东西便宜，汇率低）
        • 贸易逆差：货币流出，但物品流入，我们喜欢买别人的东西（说明我们的货币比较值钱）

      • 政府干预

        • 有的国家不干预，完全由外汇市场决定汇率：比如美国
        • 有的国家会稍微干预，但基本上也由外汇市场决定：比如中国
        • 有的国家实行固定汇率，比如港币对美元

    • 国际金融体系

      • 一开始兑换，使用金本位

        • 金本位：金作为比较的依据和价格中枢

        • 优点：

          • 各国的国际收支自动平衡
          • 即，一个国家进口的东西和出口的东西相等
          • 东西包括：商品、资本（直接投资，购买股票。。。）

      • 后来使用浮动汇率制度，即现在的外汇市场

    • 人民币汇率提高

      • 优点

        • 中国人民生活凭空提高
        • 进口为主的企业会增加利润（买国外的东西成本低了）
        • 进口增加，国内市场供应更充分，稳定国内价格

      • 缺点

        • 部分出口企业会受损
        • 引发金融系统不稳定，因为人民币升值，别人会提前买好人民币或以人民币为计价的资产，等升值了再出售，就会引起金融和资本市场的不稳定。比如股价和房价

      • 限制

        • 人民币已经实现了经常项目的自由兑换，比如想要买美国商品要兑换美元，可以直接用人民币兑换美元
        • 但还没有实现资本项目自由兑换，比如要买美国股票，或房子，想要兑换美元，人家就不给你兑换，因为中国目前不允许做这个事
